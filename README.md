# TikTok-Creative-Performance-Analysis
Analysis of 157 TikTok videos from fast food brands (KFC, McDonald's, Burger King Spain).

## Key Findings
- Brand control kills influencer performance (25x drop)
- Branding > Promo (4.6x advantage)
- Chaotic vibe dominates HIGH tier (65% vs 5% LOW)

## Dataset
- **157 videos** (Oct 2024 - Jan 2025)
- **48 variables** coded (see [GLOSSARY](data/GLOSSARY.md))
- **3 brands**: KFC, McDonald's, Burger King Spain

## Repository Structure
- `notebooks/` - Jupyter notebook with full analysis
- `data/` - Dataset + variable definitions
- `outputs/` - Figures and conclusions
- `docs/` - Methodology and findings
